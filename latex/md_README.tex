 \hypertarget{md_README_autotoc_md4}{}\doxysection{\texorpdfstring{Выбор режима программы}{Выбор режима программы}}\label{md_README_autotoc_md4}
После запуска программы на экране появится текст в вариантами выбора. После его появления нужно ввести цифру. 
\begin{DoxyCode}{0}
\DoxyCodeLine{Выберите\ действие}
\DoxyCodeLine{\ \ \ \ \ \ \ \ Решение\ квадратного\ уравнения\ (введите\ 1)}
\DoxyCodeLine{\ \ \ \ \ \ \ \ Запустить\ тест\ программы\ \ \ \ \ \ (введите\ 2)}
\DoxyCodeLine{1}

\end{DoxyCode}
 Чтобы запустить режим решения квадратного уравнения, нужно ввести 1. Чтобы запустить режим тестирования, нужно ввести 2. При любом другом значении, программа выведет ошибку. 
\begin{DoxyCode}{0}
\DoxyCodeLine{Ошибочный\ ввод}

\end{DoxyCode}
